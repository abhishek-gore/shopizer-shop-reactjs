# Shopizer Customer Shop Documentation

Complete technical documentation for Shopizer customer-facing shop (React).

## 📚 Documentation Files

- **[TECH-STACK.md](./TECH-STACK.md)** - React technology stack
- **[ARCHITECTURE.md](./ARCHITECTURE.md)** - Frontend architecture and patterns

## 🚀 Quick Start

```bash
npm install
npm start
```

Access: http://localhost:3000

## 🔑 Key Technologies

- **React 16.6**
- **Redux** + **Redux Thunk** for state management
- **React Router** for navigation
- **React Bootstrap** for UI
- **Axios** for API calls
- **Stripe** for payments

## 📖 Features

- Product browsing and search
- Shopping cart
- Checkout process
- Customer account management
- Order history
- Multi-language support (English, French)
